# Neste projeto foi feito:

1. **_Conteinerizar_** aplicações;
1. Criar uma conexão entre elas;
1. Orquestrar seu funcionamento.

Temos uma aplicação full-stack neste repositório: um **aplicativo de tarefas**! Esta aplicação precisa ser conteinerizada para funcionar. Foi desenvolvido os arquivos de configuração para cada frente específica: `Front-end`, `Back-end` e, no nosso caso, para um aplicativo de `teste` que valida se as aplicações estão se comunicando.

Foi criado as imagens para as aplicações e configurar essas imagens com o `docker-compose`.

Utilizado uma série de comandos do `docker` com diferentes níveis de complexidade.
